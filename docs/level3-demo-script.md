# Level 3 Demo Script (3-5 Minutes)

## 1. Problem Framing (30s)
Proof of Talk has 2,500 high-value participants. The challenge is not discovery volume, but high-probability deal introductions with clear rationale.

## 2. Data Ingestion + Enrichment (45s)
Show `data/test_profiles.json` and explain profile normalization + enrichment tags from `app/enrichment.py`.

## 3. Matching Engine (60s)
Show weighted scoring in `app/matching.py`:
- fit score
- strategic complementarity
- deal readiness
- weighted final ranking

## 4. Organizer Dashboard (90s)
In the web app:
- Overview metrics
- Top recommended intro pairs
- Per-profile priority list with rationale and confidence
- Action controls: approve/reject + notes

## 5. Explainability + Control Plane (45s)
Highlight that each recommendation has rationale text and organizer override. Emphasize practical human-in-the-loop workflow.

## 6. Close (30s)
Summarize business value:
- higher-value meetings
- better attendee ROI
- reusable infrastructure for future premium events.
